Website Plan for Final Project

Site Name

Name: Economic Forum Hub

Reason: This name was selected because it clearly represents a platform for discussions and resources related to economics.

Optional domain availability: https://josemarodas.github.io/wdd231/project/index.html

Site Purpose

The site aims to provide a centralized hub for individuals interested in economics. It will offer resources such as book recommendations, forums for discussion, and articles on various economic topics.

Scenarios

Scenario 1: What are the best books to understand the basics of economics?

Scenario 2: How can I join the economic discussion forum?

Color Schema

Primary Color: #1a2a4e (used for headings and background)

Secondary Color: #243b6c (used for paragraphs and accents)

Typography

Heading Font: Roboto, sans-serif (used for all headings)

Body Font: Arial, sans-serif (used for all body text)
